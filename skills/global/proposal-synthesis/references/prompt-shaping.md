# Prompt Shaping

Use `thinking-lenses` before proposal generation only when the raw ask is likely to branch too widely or hide the real objective.

## Default Choices

- `mode: design`
  Use when the user wants new options or structural alternatives.
- `mode: decide`
  Use when the user mostly wants option comparison and a recommendation.

## Delivery Mode

- `inline-structured`
  Default for medium proposal work.
- `pre-optimizer`
  Use only when the design space is expensive, high-risk, or unusually ambiguous.

## Lens Pair Suggestions

- primary: `First principles`
- support: `MECE`

Or, when the task needs more divergence pressure:
- primary: `Lateral thinking`
- support: `SCAMPER`

Keep the shaping pass shorter than the proposal-generation pass.
